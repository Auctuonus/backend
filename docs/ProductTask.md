### Челы
- авторизация через телегу

### Бабки
- у каждого чела есть кошелек с одной валютой.
- храним в виде интов (т.е. в виде минимальных единиц можно и сразу минимальные отображать)
- все махинации с бабками через транзакции и с сохранением записи в журнал

### Предметы
- предмет это просто какой то текст
- у предмета есть номер и коллекция - вместе это айди
{id: uuid, value: "lvvdsfkk", num: "123", collectioinName: "qweqw"}

### Аукционы Продавец
- можно выставлять несколько можно 1 предмет
- если предметов больше 1 то можно создавать несколько раундов
- можно настроить весь аукцион / отдельные раунды
    - минимальная ставка
    - минимальная разница в ставках
    - призы по айди либо по номерам предметов
    - anti-sniping (в том числе сколько минут прибавляется)
    - время конца (относительное / абсолютное)
- ранер
    - обрабатывает аукцион
    - проставляет завершенные раунды / аукционы
    - начинает следующие раунды
    - присваивает предметы

### Аукционы Покупатель
- можно повышать свою ставку
- ранжирование по ставке DESC
- лочить по в redis
